# Math

## Static Methods

### Sin
Returns the sine of a number in radians
- Parameters: `x: Number`
- Returns: `Number`

### Cos
Returns the cosine of a number in radians
- Parameters: `x: Number`
- Returns: `Number`

### Tan
Returns the tangent of a number in radians
- Parameters: `x: Number`
- Returns: `Number`

### Floor
Rounds the number downwards to the nearest integer
- Parameters: `x: Number`
- Returns: `Number`
